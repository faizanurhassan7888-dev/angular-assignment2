Music Scene Manager
This project is an Angular assignment based on a simple music event manager. It displays a list of bands, lets the user filter them by genre, shows upcoming performance details, and allows a band to be removed from the list.

Student ID: CC205484

Features
Displays a list of musical acts using a reusable BandCardComponent
Filters bands by genre
Shows a Sold Out badge when a band is booked
Uses different visual styles for Metal and Rock bands
Changes card border color based on ticket price
Uses Angular pipes for uppercase text, date formatting, and currency formatting
Uses parent-to-child and child-to-parent communication with @Input() and @Output()
Technologies Used
Angular
TypeScript
HTML
CSS
Project Structure
Band interface is stored in a separate file
BandListComponent works as the parent component and stores the band data
BandCardComponent works as the child component and displays each band card
